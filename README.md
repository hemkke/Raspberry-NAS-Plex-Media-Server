# Raspberry-NAS-Plex-Media-Server
Transformă un Raspberry Pi într-un NAS personal și server media cu Plex. Acest proiect include pași detaliați de instalare, configurații pentru Samba/NFS, fișiere Docker pentru Plex și sfaturi pentru gestionarea fișierelor media. Ideal pentru cine vrea să-și creeze propriul server de stocare și streaming la domiciliu.
